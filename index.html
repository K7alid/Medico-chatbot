Medico(Chatbot): Medical Query Assistant

Abstract

Medico is a Medical Query Assistant application designed to process and respond to user queries related to health and medical topics. It leverages machine learning models to provide accurate and helpful responses, supporting both text and audio inputs. The application integrates multiple response sources, including Direct Model, ChromaDB, and Retrieval-Augmented Generation (RAG), and offers text-to-speech capabilities for enhanced user interaction.

Features

  • Text and Audio Input
  • Multiple Response Sources: Direct Model, ChromaDB, RAG
  • Text-to-Speech Conversion
  • User Interface with Gradio

Models Used

  • For Embedding: sentence-transformers/all-MiniLM-L12-v2
  • For Generation: Australia/Mixtral-8x7B-Instruct-v0.1
  • For Speech to Text: open/whisper-base.en
  • For Text to Speech: gTTS library
